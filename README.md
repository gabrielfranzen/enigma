## Sobre o projeto

Este projeto é um enigma interativo feito para diversão, com um tema misterioso.

## Sobre a experiência

A proposta é explorar pistas, testar hipóteses e avançar por etapas até revelar a cena final.
O foco é a curiosidade, a interpretação dos sinais e a criatividade de quem joga.

## Recomendação importante

Para preservar a experiência do enigma, é recomendado **não utilizar agentes de IA** para decifrar as respostas automaticamente.

Se você quiser uma ajuda, pode usar o **ChatGPT**.

## Aviso

Este projeto foi pensado para entretenimento e desafio técnico leve.
Divirta-se investigando.
